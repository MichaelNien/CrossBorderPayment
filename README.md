## 信用卡清算機制

### Visa

Visa全球通匯(Visa Direct)

### Mastercard

全球跨境匯款技術解決方案(Mastercard Send XB)

- 帳戶 對 帳戶 匯款
- 帳戶 對 錢包 匯款
- 帳戶 對 信用卡卡號 匯款
- 帳戶 對 據點 匯款
